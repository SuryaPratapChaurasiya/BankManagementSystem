# **Bank Management System**

## **Description**
A simple console-based banking system using **Core Java** and **File I/O**. Users can create accounts, deposit, withdraw, check balances, and manage their transactions. The system persists data in a file (`accounts.txt`), allowing users to maintain their banking information even after the application is closed.

## **Features**
- **Create New Account**: Allows users to create a new bank account.
- **Deposit and Withdraw Money**: Users can deposit and withdraw money from their accounts.
- **Balance Inquiry**: Allows users to check their current balance.
- **Transaction History**: Keeps track of deposits and withdrawals.
- **Data Persistence**: The system saves account data to a file (`accounts.txt`), ensuring data is retained across application runs.
