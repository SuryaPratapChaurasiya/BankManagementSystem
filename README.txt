# Bank Management System 

## Description
A simple console-based banking system using Core Java and File I/O. Users can create accounts, deposit, withdraw, and check balance.

## Features
- Create New Account
- Deposit and Withdraw Money
- Balance Inquiry
- Data Persistence using File I/O

## How to Run
1. Compile the code:
   javac BankAccount.java BankSystem.java
2. Run the application:
   java BankSystem
3. Data will be saved in `accounts.txt`

## Files
- BankAccount.java
- BankSystem.java
- accounts.txt (auto-created)
